# MoodleReports

Программа для извлечения данных о попытках студентов из резервных копий Moodle (.mbz) и создания индивидуальных HTML-отчётов.

## Возможности

- Извлечение данных из backup-архивов Moodle (.mbz)
- Создание отдельного HTML-отчёта для каждого студента
- Отображение правильности каждого ответа
- Встраивание изображений в отчёты
- Фильтрация правильных/неправильных ответов

### Использование

1. **Запустите** `MoodleReports.exe`
2. **Выберите** архив Moodle (.mbz) - кнопка "Открыть"
3. **Выберите** папку для сохранения - кнопка "Выбрать"
4. **Нажмите** "Выгрузить"
5. Результаты будут в выбранной папке

### Опции

- **Чекбокс "Отображать правильные попытки"** (по умолчанию включён):
  - Включён - показываются все вопросы
  - Выключен - показываются только неправильные ответы

### Поддерживаемые типы вопросов

| Тип | Описание |
|-----|----------|
| multichoice | Выбор одного ответа |
| multianswer | Cloze - несколько ответов |
| shortanswer | Текстовый ответ |
| match | Сопоставление |
| gapselect | Заполнение пропусков |

### Формат отчёта

Каждый отчёт - отдельный файл `Фамилия_Имя_moodle_backup.html`

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| Архив не открывается | Используйте файл .mbz Moodle |
| Отчёты не создаются | Проверьте наличие quiz.xml и questions.xml |
| Пустые отчёты | Проверьте наличие попыток пользователей |

## Технические детали
