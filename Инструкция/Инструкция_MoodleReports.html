Инструкция по использованию MoodleReports

Программа MoodleReports извлекает данные о попытках из резервных копий Moodle (.mbz) и создаёт индивидуальные HTML-отчёты для каждого студента.

Скачать программу

Что умеет программа

  • Извлекать данные о попытках из backup-архивов Moodle
  • Создавать отдельный HTML-отчёт для каждого студента
  • Показывать правильность каждого ответа
  • Встраивать изображения в отчёты
  • Фильтровать правильные/неправильные ответы

Шаг 1 - Запуск и интерфейс

1

Скачайте программу по кнопке выше или запустите MoodleReports.exe из папки. Откроется главное окно программы:

В окне есть:

  • Кнопка "Открыть" - для выбора файла резервной копии Moodle (.mbz)
  • Кнопка "Выбрать" - для выбора папки, куда сохранить отчёты
  • Кнопка "Выгрузить" - запуск экспорта
  • Чекбокс "Отображать правильные попытки" - фильтр для отображения
Как работает фильтр: Если галочка стоит - в отчёте показываются все вопросы. Если галочку снять - показываются только вопросы с неправильными ответами (балл меньше 99%). Это удобно, чтобы студенты видели только свои ошибки.

Шаг 2 - Выгрузка отчётов

2

Нажмите "Выгрузить". Программа распакует архив, обработает данные и создаст отчёты. После завершения появится сообщение:

В сообщении указывается путь к папке с отчётами и количество созданных файлов.

Шаг 3 - Результаты в папке

3

Откройте папку, которую выбрали для сохранения. Там будут файлы:

Каждый файл именуется по ФИО студента: Иванов_Иван_moodle_backup.html

Откройте любой файл в браузере - там будет подробная информация о попытках.

Частые проблемы и решения

ПроблемаРешение
Архив не открываетсяУбедитесь, что это файл .mbz Moodle
Отчёты не создаютсяПроверьте наличие quiz.xml и questions.xml в архиве
Пустые отчётыПроверьте, что есть попытки пользователей
Ошибка изображенийИзображения должны быть в папке files внутри архива
Важно: Программа работает только с архивами Moodle (.mbz). Обычные ZIP-архивы не подойдут.
